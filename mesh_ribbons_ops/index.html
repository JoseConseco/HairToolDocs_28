Mesh Ribbon operations:¶

Add Ribbon Weights¶

Add Ribbon Weights - create vertex weight gradient from strand root to tip

Add vertex colors gradient¶

Add vertex colors gradient - create black and white color gradient from strand root to tip

Add random vertex colors¶

Add random vertex colors - assign different color for each mesh ribbon strand.

I recommend Vertex Color Master addon for more detailed control over managing vertex colors on mesh (I added some features to VCMaster with Hair Tool in mind - more advanced color randomization per mesh island, drawing gradients)
